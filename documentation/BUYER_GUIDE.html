Timeout HRM

Buyer Guide

Premium full-stack HRM SaaS template built with Next.js, Express, Prisma, PostgreSQL, Socket.IO, and a polished admin dashboard experience.

Package Contents

  • frontend/ Next.js application
  • backend/ Express + Prisma API
  • documentation/ setup, deployment, troubleshooting and support guides
  • INSTALLATION.md quick start instructions
  • .env.example workspace reference

Requirements

  • Node.js 20.x to 24.x
  • npm 10+
  • PostgreSQL 14+
  • A deployed backend URL for production frontend builds

Quick Start

  1. Create environment files from the included templates. 
    cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env.local
  2. Install dependencies from the workspace root. 
    npm install
  3. Run database migrations and seed demo data. 
    npm run db:migrate -w backend
npm run seed -w backend
  4. Start both applications. 
    npm run dev

Production Setup Notes

  • Set NEXT_PUBLIC_API_URL in frontend/.env.local or your host environment.
  • Set a production DATABASE_URL and secure auth secrets in backend/.env.
  • Update CLIENT_ORIGINS in the backend env to match your deployed frontend domain.
  • SMTP and AI provider values are optional and can be enabled later.
